Yankees Clicky Game

How good is your memory?

A web application game created with React. The app begins by rendering images of Yankees players randomly to the screen and keeps track of the user's score. The user's score is incremented when clicking an image for the first time.

The user's score is reset to 0 if they click the same image more than once. Every time an image is clicked, the images rendered to the page shuffles themselves in a random order.

Once the user's score is reset after an incorrect guess, the game restarts.

  • Date: September 2018
    Technologies used:
  • React

HIMYM Trivia

It's legen ... wait for it ... dary!


This is a trivia game app of the TV show "How I Met Your Mother." The app will show only one question until the player answers it or their time runs out.

If the player selects the correct answer, a screen is shown congratulating them for choosing the right option, along with a gif. After a few seconds, the next question will be displayed without user input. The scenario is similar for wrong answers and time-outs.

If the player runs out of time, the player will be told that time's up and display the correct answer, along with a gif. Wait a few seconds, then show the next question.

If the player chooses the wrong answer, the player will be told they selected the wrong option, along with a gif, and display the correct answer. Wait a few seconds, then show the next question.

On the final screen, show the number of correct answers, incorrect answers, and an option to restart the game (without reloading the page).

  • Date: June 2018
    Technologies used:
  • HTML
  • CSS
  • JavaScript
  • Bootstrap

MLB All-Star Hangman

Hit it out of the park!

This is a Hangman app built with the theme of MLB All-Stars. Once the user chooses to start the game, a randomly selected All-Star's name will appear on the screen in dashes. If the user guesses a letter correctly, the corresponding dash will turn to the letter. If the user guesses incorrectly, the guessed letter will be added to the list of incorrect letters.

With each incorrect guess, the amount of guess left will decrease. If it hits zero, the user loses. If the user guesses the name correctly, an alert window will pop up to congratulate the user, and a score will be tallied toward the win column.

  • Date: June 2018
    Technologies used:
  • HTML
  • CSS
  • JavaScript
  • Bootstrap
  • Font Awesome

Liri Node App

At your command


LIRI is like iPhone's SIRI. However, while SIRI is a Speech Interpretation and Recognition Interface, LIRI is a Language Interpretation and Recognition Interface. LIRI will be a command line node app that takes in parameters and gives client back data.

What each command does
  1. node liri.js my-tweets
    • This will show the chosen account's last 20 tweets and when they were created at in the terminal/bash window.
  2. node liri.js spotify-this-song "song name here"
    • This will show the following information about the song in your terminal/bash window:
      • Artist(s)
      • The song's name
      • A preview link of the song from Spotify
      • The album that the song is from
  3. node liri.js movie-this "movie name here"
    • This will output the following information to your terminal/bash window:
      • Title of the movie
      • Year the movie came out
      • IMDB Rating of the movie
      • Rotten Tomatoes Rating of the movie
      • Country where the movie was produced
      • Language of the movie
      • Plot of the movie
      • Actors in the movie
  4. node liri.js do-what-it-says
    • Using the fs Node package, LIRI will take the text inside of random.txt and then use it to call one of LIRI's commands.
  • Date: May 2018
    Technologies used:
  • Node.js
  • Dotenv
  • Request
